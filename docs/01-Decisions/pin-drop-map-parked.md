# PUBLIC PRECISE PIN-DROP MAP — PARKED POST-LAUNCH

Status: locked

Source: Session Log — 21 July 2026 (location/address design)

A public precise pin-drop map (WhatsApp-style manual placement) is explicitly PARKED as post-launch.

Research done in-session: the competitor's site and RoomKing (SA's largest backyard-room platform, active in the actual target informal-settlement market) both use suburb/area text only, no map pins, and both operate successfully at scale on that model.

Decision: suburb/area-level display is sufficient for launch. A real interactive map remains a genuine differentiator worth building properly post-launch, not squeezed into the pre-launch window.
