# FILE NAMING — NO SUFFIX / -spa SUFFIX

Status: locked

Source: Session Log — 14 July 2026 (LOCKED naming decision)

Per-supplier-type files use the register.html / register-spa.html convention: **no suffix for therapist, `-spa` suffix for spa**.

- `dashboard.html` (therapist) / `dashboard-spa.html` (spa)
- `gallery.html` (therapist, built) / `gallery-spa.html` (spa, future — mirrors gallery.html once it is fully solid, the same way M2/M4 followed M1/M3)

Do NOT use short-code names like "T-Dashboard" / "S-Dashboard".
