💆 MassageMap — Sitemap

Screen flow, filenames, and URL params. Not customer-facing.

🏠
Home
index.html

Province grid — 9 province cards in 2-column layout. Search bar filters suburbs across all provinces.

URL params in: none
?province=Gauteng
📍
Suburb List
suburb.html

All suburbs for the selected province, each showing a therapist count. Tapping a suburb goes to listings.

URL params in: province
Back to: index.html
?suburb=Sandton&province=Gauteng
👥
Therapist Listings
listings.html

Compact table view of therapists in a suburb. Shows name, classification, and primary service per row. Adult-classified therapists are never shown.

URL params in: suburb, province
Back to: suburb.html?province=…
?id=1&suburb=Sandton&province=Gauteng
👤
Therapist Profile
profile.html

Full profile: name, classification, suburb, price, services, massage types, availability, mobile massage. Send Enquiry CTA button.

URL params in: id, suburb, province
Back to: listings.html?suburb=…&province=…
